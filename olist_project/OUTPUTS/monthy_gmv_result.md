# monthly_gmv

## 1. 分析口径

- 订单范围：`delivered`
- 金额口径：`SUM(payment_value)`
- 时间口径：`order_purchase_timestamp` 所在月份

说明：

- `2016-10`、`2016-12` 为早期零散月份，不建议参与正常经营期横向比较。
- 主体观察区间建议聚焦 `2017-01` 到 `2018-08`。

## 4. 增长驱动观察

GMV 的主要波动来自订单数和活跃买家数变化，而不是复购频次变化：

- `orders_per_buyer` 长期稳定在 `1.01~1.02`
- `AOV` 有波动，但不是大盘增长的核心驱动
- `2017-11` GMV 环比增加 `402,387.78`，其中订单数增加 `2,811`，活跃买家增加 `2,766`

## 5. 阶段性结论

- 平台在 `2017-01` 到 `2017-11` 经历了明显扩张，增长主要来自新增买家和订单量提升。
- `2017-11` 是全周期最强增长点，但客单价并未同步上升，说明增长更偏流量和订单扩张。
- `2018-04` 到 `2018-07` GMV稳定在 `100万+` 区间，平台进入相对成熟阶段。
- `2018-08` 订单数继续增加，但客单价回落到 `155.16`，导致 GMV 较 `2018-07` 小幅下滑。
